The Order of API listed below is Important. /createRoom
returns room_id of new room

/addUser/room_id
returns user_id of new user added to room room_id

/closeRoom/room_id
deletes room room_id if created, then returns status

/getUsers/room_id
get list of user_id in room_id

/startPlayer/room_id
start player in room_id

/seekTo/room_id/seek_time
seek to seek_time(seconds) in room_id

/getCurrentSecond/room_id
get current time in room_id
